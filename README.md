# OPERATION: PROM

A retro 8-bit adventure game disguised as a promposal. Five levels, inside jokes, a fleeing "no" button, mango confetti, and a downloadable virtual corsage.

## Play

**[ranveerd11.github.io/prom](https://ranveerd11.github.io/prom/)**

## Levels

| # | Room | Challenge |
|---|------|-----------|
| 1 | The Google Meet | Join the call |
| 2 | The Sleep Schedule | Identify the true superpower |
| 3 | The Guest List | Pick the sweetest guest |
| 4 | The Late Night Call | Find the hidden letter |
| 5 | The Big Question | Try clicking "no" |

## Tech

Single HTML file. No frameworks, no build tools, no dependencies. Just `index.html` and a few images.

- Canvas-drawn corsage with PNG download
- Web Audio API for 8-bit sound effects
- Physics-based fleeing button with `requestAnimationFrame`
- Press Start 2P pixel font
